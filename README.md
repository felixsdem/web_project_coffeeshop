# Triple Coffee

## Descripción del proyecto

Triple Coffee es una página web estática para una cafetería ficticia especializada en café de especialidad.  
El sitio presenta información sobre el concepto del local, recetas, un formulario de reserva y enlaces a redes sociales.

El objetivo del proyecto es aplicar buenas prácticas de maquetación y organización de estilos utilizando la metodología BEM, así como reforzar el uso de HTML semántico y CSS modular.

### Tecnologías utilizadas

- HTML5 semántico
- CSS3
- Metodología BEM
- Organización de estilos por bloques (`blocks/`)
- Normalize.css para consistencia entre navegadores
- Git y GitHub para control de versiones
